# [Client Name] Website

Built with Astro + Tailwind. Deployed via Coolify on dunne.design hosting infrastructure.

## Local development

```bash
npm install
npm run dev
```

Site will be live at `http://localhost:4321`.

## Deployment

Push to `main` — Coolify rebuilds and deploys automatically. Live URL: see project admin.

## Project structure

- `src/pages/` — page routes (each `.astro` file becomes a page)
- `src/components/` — reusable components
- `src/layouts/` — page layouts
- `public/` — static assets (images, fonts, favicon)
